**Basics** <br>
Principle - Last In First Out (LIFO) <br>
Basic Terminologies - Top and Size, Push(Insertion) and Pop(Deletion) happens at top <br>
Types - Fixed Size and Dynamic Size (Implemented using Linked List and ArrayList in Java) <br>
Common Operations - push(), pop(), top(), isEmpty(), size() <br><br>

Implementation can be done using Aray, LinkedList and Deque
